01Filosofia de estudo

O guia é construído em torno de quatro princípios. Eles existem para evitar o erro mais comum de quem estuda back-end: confundir "ler conteúdo" com "saber programar".

// 01

// 02

// 03

// 04

enum OrderStatus: string {
    case Pending  = 'pending';
    case Paid     = 'paid';
    case Shipped  = 'shipped';

    public function canTransitionTo(self $next): bool {
        return match ([$this, $next]) {
            [self::Pending, self::Paid],
            [self::Paid, self::Shipped] => true,
            default => false,
        };
    }
}

interface DiscountStrategy {
    public function apply(Money $total): Money;
}

final readonly class PercentOff implements DiscountStrategy {
    public function __construct(private int $percent) {}

    public function apply(Money $total): Money {
        return new Money(
            (int) ($total->cents * (100 - $this->percent) / 100),
            $total->currency,
        );
    }
}

    private array $events = [];

    public function pay(Money $amount): void {
        if (!$this->status->canTransitionTo(OrderStatus::Paid)) {
            throw new DomainException('invalid transition');
        }
        $this->status = OrderStatus::Paid;
        $this->events[] = new OrderPaid($this->id, $amount);
    }

    public function pullEvents(): array {
        return $this->events;
    }

final class IdempotencyMiddleware implements MiddlewareInterface {
    public function __construct(private CacheInterface $cache) {}

    public function process(
        ServerRequestInterface $req,
        RequestHandlerInterface $next,
    ): ResponseInterface {
        $key = $req->getHeaderLine('Idempotency-Key');
        if ($key && $cached = $this->cache->get($key)) {
            return $cached;
        }
        $res = $next->handle($req);
        if ($key) $this->cache->set($key, $res, 86400);
        return $res;
    }
}

03Exemplos-chave

Snippets que aparecem repetidas vezes no guia — vale memorizar a estrutura, não a sintaxe.

Property hooks (PHP 8.4)

final class User {
    public string $email {
        set(string $value) {
            if (!filter_var($value, FILTER_VALIDATE_EMAIL)) {
                throw new InvalidArgumentException('invalid email');
            }
            $this->email = strtolower($value);
        }
    }
}

Asymmetric visibility (PHP 8.4)

final class Counter {
    public private(set) int $value = 0;

    public function increment(): void {
        $this->value++;
    }
}

Atributo customizado + Reflection

#[Attribute(Attribute::TARGET_PROPERTY)]
final readonly class MaxLength {
    public function __construct(public int $length) {}
}

final class CreateUser {
    #[MaxLength(120)]
    public string $name;
}

$ref = new ReflectionProperty(CreateUser::class, 'name');
foreach ($ref->getAttributes(MaxLength::class) as $attr) {
    $rule = $attr->newInstance();
    // $rule->length === 120
}
